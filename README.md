## PROGRAM:
```
<!DOCTYPE html>
<html>
<head>
 <title>Using Python's SimpleHTTPServer Module</title>
 <style>
 #rectangle {
 height: 50px;
 width: 100px;
 background-color: #00f28f;
 }
 </style>
</head>
<body>
 <h2>Rectangle served by SimpleHTTPServer</h2>
 <div id="rectangle"></div>
</body>
</html>
```
